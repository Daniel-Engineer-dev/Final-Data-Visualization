# Data Dictionary

Tài liệu này sẽ được hoàn thiện sau spike dữ liệu.

| Biến | Kiểu | Đơn vị | Nguồn | Mô tả |
|---|---|---|---|---|
| date | date | ngày | Open-Meteo | Ngày quan sát |
| location | string | - | Cấu hình dự án | Tên địa điểm đại diện |
| latitude | float | độ | Open-Meteo/cấu hình | Vĩ độ |
| longitude | float | độ | Open-Meteo/cấu hình | Kinh độ |
| temperature_2m_mean | float | °C | Open-Meteo | Nhiệt độ trung bình ngày |
| precipitation_sum | float | mm | Open-Meteo | Tổng lượng mưa ngày |
| wind_speed_10m_max | float | km/h | Open-Meteo | Tốc độ gió cực đại ngày |
