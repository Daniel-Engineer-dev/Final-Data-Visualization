## 7. Module AI đề xuất

### Giá trị thực sự của AI

> AI giúp người dùng thực hiện phân tích mới nào mà dashboard cố định không thể cung cấp?

### Luồng AI bắt buộc

Xác nhận đề tài hỗ trợ toàn bộ luồng:

- [ ] AI nhận yêu cầu và schema/ngữ cảnh dữ liệu.
- [ ] AI sinh code hoặc truy vấn phân tích.
- [ ] Code được hiển thị rõ ràng.
- [ ] AI giải thích code bằng ngôn ngữ tự nhiên.
- [ ] Người dùng có thể chỉnh sửa hoặc từ chối.
- [ ] Code ở trạng thái chờ cho đến khi được phê duyệt.
- [ ] Code chỉ chạy local sau khi người dùng phê duyệt.
- [ ] Chỉ cho phép thao tác đọc dữ liệu.
- [ ] Dữ liệu gốc không bị thay đổi.
- [ ] Prompt, code, quyết định, kết quả và giải thích được lưu log.
